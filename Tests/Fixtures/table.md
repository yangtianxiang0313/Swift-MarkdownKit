# 表格测试

## 基础表格

| 列一 | 列二 | 列三 |
|------|------|------|
| A1   | A2   | A3   |
| B1   | B2   | B3   |
| C1   | C2   | C3   |

## 对齐表格

| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|-------:|
| 内容1  | 内容2    | 内容3  |
| 较长的内容 | 中 | 短 |

## 带格式表格

| 功能 | 描述 |
|------|------|
| **加粗** | 使用 `**text**` |
| *斜体* | 使用 `*text*` |
| `代码` | 使用反引号 |
